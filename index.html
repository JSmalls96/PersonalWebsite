C  OpenSSL
RSA Encryption & Digital Signature
Program which demonstrates each step of the RivestShamirAdleman
(RSA) public-key cryptosystem which is commonly used for secure
communications.

The application generates two large random prime numbers, and then
uses them to generate public and private key pairs, which are then
used to do encryption, decryption, digital signature generation,
and digital signature verification.

My implementation was done in C, and utilizes the OpenSSL library.
